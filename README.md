# ⚽ AI футбол — ЧМ-2026

Браузерная футбольная игра с настоящими сборными и игроками чемпионата мира 2026 (США · Канада · Мексика). Некоммерческий фанатский проект в ознакомительных целях.

## Как запустить локально

Просто откройте файл `index.html` двойным кликом — игра работает в любом современном браузере без установки чего-либо.

## Как играть

1. **Начать игру** → выберите свою команду → выберите соперника.
2. Вы всегда управляете игроком с **жёлтой стрелкой** — тем, кто ближе всех к мячу (как в FIFA).

| Клавиша | Действие |
|---|---|
| WASD / стрелки | движение |
| Shift | ускорение (тратит выносливость) |
| Пробел | пас (в сторону движения) |
| X | удар по воротам |
| C | пас вразрез (на ход нападающему) |
| E | сменить игрока (в обороне) |
| P / Esc | пауза |

В игре работают настоящие правила: ауты, угловые, удары от ворот, офсайд (можно выключить в настройках), два тайма по 45 минут на табло.

## Команды

12 настоящих сборных ЧМ-2026: все 8 участников 1/4 финала (Аргентина, Франция, Испания, Англия, Норвегия, Бельгия, Марокко, Швейцария) плюс Бразилия, Португалия, США и Мексика. У каждой — реальный стартовый состав с настоящими номерами, своя форма и свои сильные стороны, основанные на реальных выступлениях на турнире.

## Как выложить на GitHub Pages

1. Зарегистрируйтесь/войдите на [github.com](https://github.com).
2. Нажмите **+** (вверху справа) → **New repository**. Назовите, например, `ai-football`, выберите **Public**, нажмите **Create repository**.
3. На странице репозитория нажмите **uploading an existing file** (или **Add file → Upload files**).
4. Перетащите в окно ВСЁ содержимое этой папки: `index.html`, `README.md` и папки `css`, `js` (папки можно перетаскивать целиком). Нажмите **Commit changes**.
5. Откройте **Settings** (вкладка репозитория) → слева **Pages**.
6. В разделе **Build and deployment** → Source: **Deploy from a branch** → Branch: **main**, папка **/ (root)** → **Save**.
7. Через 1–2 минуты игра будет доступна по адресу:
   `https://ВАШ_ЛОГИН.github.io/ai-football/`
   Эту ссылку можно отправлять кому угодно.

## Технологии

Чистый HTML + CSS + JavaScript (Canvas 2D, Web Audio API). Без библиотек, без сборки, без сервера — поэтому работает и локально, и на GitHub Pages.
